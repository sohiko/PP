プログラミングとは？

プログラミングは、コンピュータに特定のタスクや動作を指示するための手順を作ることです。
プログラミング言語を使って、ウェブサイトやアプリ、ゲームなどを作成することができます。
プログラミングは、コンピュータに自分の考えを伝える方法であり、創造的な問題解決や論理的思考を駆使する活動でもあります。
難しく聞こえますが、ある程度慣れれば簡単に扱え、習得にもあまり時間がかかりません。簡単な英語が使えれば支障はないでしょう。

パソコン、スマホを開いた先にあるものは全てプログラミングによって作られています。

種類

　

HTML

ウェブサイトの構築に使われ、すべてのウェブサイトで使われています。
　

CSS

ウェブサイトのデザインに使われ、すべてのウェブサイトのデザインに使われています。
　

JavaScript

ウェブサイト等に簡単に動きをつけることができ、タイマーや時間表示、計算などができます。
　

PHP

サーバーサイドで使われる言語であり、ユーザー側が見ることはありません。会員登録処理や、パスワードなどの保存が可能です。
　

Java

今注目されている言語であり、WEB開発、アプリケーション開発が可能で、様々な開発に使用することができます。
　

Python

今注目のAIなど、人工知能にまつわる言語で、機械学習を主としています。また、構文が簡単という特徴もあります。

プログラミングはプログラミング言語を使って行います。プログラミング言語の代表格を6つ、独断と偏見で取り上げました。

プログラミング言語にはWEB開発、アプリ開発、AI開発、サーバー操作、OS操作など様々な用途に合ったものが存在し、200〜300もの言語が存在していると言われています。しかし、その中でも使われているのは10％ほどです。

皆さんもお気に入りの言語を見つけてみましょう。

歴史

プログラミングの歴史について簡単にご紹介します

1940年代

機械言語でコンピューターを動かすことができるようになり、単純な処理が可能に

1960年代

手続き型言語が誕生。事務処理の効率化が図られる。（日本ではCOBOLが誕生）

1970年代

データベース（登録情報などを保存する場所）を操作する言語、SQLが誕生

1980年代

C言語が中心となる。C++も登場するものの、当時はオブジェクト指向が普及しておらいなかった。

1990年代前半

Pythonが開発され、広く普及する。Microsoft社がVBAを開発。Excelに搭載され、多くに人が利用する。

1990年代後半

日本初のプログラミング言語、Rubyを発表。国内外で人気になる。
同じ年、Javaが発表され、C言語などオブジェクト指向が主流ではなかったプログラミング言語が一転し、オブジェクト指向を使った言語が取り入れられるように。Javaも瞬く間に普及。
さらに同じ年、現在のPHPの前身である PHP/FI が開発。
その2年後、JavaScriptが開発され、ウェブサイトのクライアント側（閲覧者側）で簡単に動的処理が可能に。

2000年代

C++にJavaを盛り込んだC＃が発表される。
また、大手Google社がプログラミング言語、Goを発表。

2010年代

クライアントサイド、サーバーサイドの両方で機能するDartがGoogle社から発表される。また、Appleによってアプリの開発に使われる言語、Swiftが発表される。
大手Facebook社からもPHPを基に開発されたHackが発表される。
